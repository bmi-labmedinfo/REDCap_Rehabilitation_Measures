# REDCap Clinical Scales - Rehab

Here you can find several REDCap instruments implementing validated clinical scales, including:

* NIHSS Scale
* BRASS
* Cognitive Reserve (CRIQ)

etc. 

These scales are used in the RCT StrokeFit4, recruiting post-acute stroke patients. Protocol described in [1].
Complete explanation of each scale is reported in the paper [2]

## Usage
Download the zip of the scales you're interested in and simply import it in your REDCap project.
